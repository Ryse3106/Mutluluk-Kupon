# Mutluluk Kuponu (PWA)

## iPhone'da App gibi kurulum (ikon + splash + offline)
1) Bu klasörü bir web adresine yayınla (en kolay: GitHub Pages / Netlify).
2) iPhone'da Safari ile adresi aç.
3) Paylaş (⬆️) > "Ana Ekrana Ekle".

## Önerilen yayınlama (GitHub Pages)
- GitHub'da yeni repo aç
- Bu klasördeki tüm dosyaları root'a yükle
- Settings > Pages > Deploy from branch (main / root)
- Sana bir URL verir: https://kullaniciadi.github.io/repo/

Not: Offline çalışma için ilk açılışta internet gerekir (cache dolsun diye). Sonra uçak modunda bile açılır.
